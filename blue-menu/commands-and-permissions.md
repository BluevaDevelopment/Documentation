# Commands and Permissions

**Note:** `[]` Required --- `()` Optional

| Command | Permission | Description |
|---------|-----------|-------------|
| `/bm` | `bluemenu.info` | Displays plugin information. |
| `/bm help` | `bluemenu.help` | Shows available player commands. |
| `/bm open [java/bedrock/auto] [menu] (player)` | `bluemenu.open`<br>`bluemenu.open.others` (for player arg) | Open a menu. |
| `/bm list [java/bedrock]` | `bluemenu.list` | List of available menus. |
| `/bm reload` | `bluemenu.reload` | Reload the configuration and menus. |
| `/bm editor` | `bluemenu.editor` | Open the web editor. |
| `/bm confirm [verificationId]` | `bluemenu.editor` | Confirm a web editor session. |
| `/<menu open command>` | `-` | Opens a menu with a custom command (assigned in the menu file) |

### Command Aliases
- `/bluemenu`
- `/bm`
- `/menu`
- `/menus`
